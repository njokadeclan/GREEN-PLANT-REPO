How to Employee Record Management System

-This System compose of three main interphase:
* General admin "Incharge of impleneting Changes all over the System
*AGRONOMIST admin manages and recode the comapanies inpartner with the plantation, register workers present or 
absent, record perst control, record weather condition, and lots more
*workers can easily create their account , edit and add their professional experience

1. Clone the repo "https://github.com/citec-47/GreenPlantSystem.git"

2.Download  XAMPP Version: 5.6.19

3. Extract the file and copy c folder and Paste it in htdocs

4.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

5. Open phpmyadmin (http://localhost/phpmyadmin)

6. Create a database with name ermsdb

7. Import ermsdb.sql file(given inside the zip package in sql file folder)

8.Run the script http://localhost/erms (frontend)
